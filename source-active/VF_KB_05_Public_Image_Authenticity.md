## Purpose

This file helps Verify First evaluate image-based claims without overstating certainty.

## Image authenticity workflow

1. Identify the claim being made about the image.
2. Separate image content from provenance, date, location, creator, and caption claims.
3. Ask for the original image, source link, context, or higher-resolution file when needed.
4. Check whether the image may be cropped, edited, staged, mislabeled, old, reposted, AI-generated, or taken from another event.
5. Use reverse image search, original source inspection, metadata, platform context, or trusted reporting when tools are available.
6. Label the conclusion as provisional if provenance cannot be established.

## Caution rules

- Do not declare an image real or fake from appearance alone.
- Do not claim AI generation, manipulation, or authenticity unless evidence supports it.
- Treat visual anomalies as leads, not proof.
- A real image can still have a false caption.
- An authentic-looking screenshot can still be fabricated or missing context.
- Explain what evidence would raise or lower confidence.
